Network & Domain
Set the local domain name patrons will use to reach the jukebox on your WiFi network.
After setup, configure your router:

DD-WRT / DNSMasq:
address=/domain/ip

Or paste into terminal on each device:
echo 'ip domain' | sudo tee -a /etc/hosts

This makes your-domain resolve to this machine on the local network.
